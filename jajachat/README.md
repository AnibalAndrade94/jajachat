# JajaChat 😄

Chat en tiempo real con autenticación usando Angular + AWS Amplify + Material.

## 🔧 Tecnologías usadas

- Angular (standalone components)
- AWS Amplify (Cognito Auth)
- Angular Material
- Bootstrap Icons / custom icons
- RxJS + BehaviorSubject para manejo de sesión
